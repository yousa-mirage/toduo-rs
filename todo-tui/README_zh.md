**ToDuo** 的终端版本，专为键盘效率和极客打造。

![ToDuo TUI](../.github/tui-preview.png)

## 🎮 键盘操作

TUI 采用了 Vim 风格的键位设计，让双手无需离开主键盘区。

| 键位 | 功能 | 说明 |
| :--- | :--- | :--- |
| **导航** | | |
| `j` / `↓` | 向下移动 | 选择下一个任务 |
| `k` / `↑` | 向上移动 | 选择上一个任务 |
| `g` | 跳转顶部 | 快速回到列表首项 |
| `G` | 跳转底部 | 快速到达列表末项 |
| `Tab` | 切换焦点 | 在侧边栏和任务列表间切换 |
| **任务操作** | | |
| `a` | 添加任务 | 打开添加任务面板 |
| `Space` | 切换状态 | 标记任务完成/未完成 |
| `d` | 删除任务 | 删除当前选中的任务 |
| `1` - `5` | 设置优先级 | 设置 A/B/C/D/E 优先级 |
| `0` | 清除优先级 | 移除任务的优先级 |
| `Enter` | 切换状态 | 同 Space |
| **其他** | | |
| `r` | 刷新 | 重新加载 todo.txt 文件 |
| `?` | 帮助 | 显示/隐藏帮助面板 |
| `q` | 退出 | 退出程序 |

## 🖱️ 鼠标支持

虽然是终端程序，但 `td` 对鼠标提供了极佳的支持：

* **拖拽调整侧边栏**: 鼠标按住侧边栏分割线并拖动，可自由调整侧边栏宽度。
* **点击选择**: 直接点击列表项进行选择。
* **双击编辑**: 双击任务可进入编辑模式。
* **点击切换**: 点击任务前的 `[ ]` 或 `[x]` 可直接切换完成状态。

## 🖥️ 界面布局

* **左侧**: 侧边栏。包含过滤器（所有任务、今天、未来7天）和优先级快速筛选。
* **右侧**: 任务列表。显示任务详情、优先级颜色、截止日期。
* **底部**: 状态栏。显示当前加载的文件路径和操作提示。
